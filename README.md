# RDASApp
Repository for JEDI-based data assimilation for the Rapid Refresh Forecast System (RRFS)

### Currently supported platforms
-   NOAA RDHPCS Hera
-   NOAA RDHPCS Jet
-   NOAA RDHPCS Orion
-   NOAA RDHPCS Hercules
